Project Overview

This project analyzes a 2022 sample of the World Development Indicators (WDI) dataset using Quarto and Python.

The analysis focuses on three key indicators:

GDP per capita

Life expectancy

Unemployment rate

The report includes:

Exploratory data analysis (EDA)

Descriptive statistics

Two types of visualizations (scatter plot and bar chart)

Cross-referenced figures and tables

Bibliography using BibTeX

HTML and PDF outputs

The presentation was created using Quarto with Revealjs, including a custom theme and slide transitions.
